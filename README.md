# paintball
Godot made paintball boomer shooter inspired by Ultrakill

This project is mostly for learning and portfolio purposes.

# Godot

- Engine Version: 4.6.1
- Project path: ```game/```

# Obsidian Vault

The [Obsidian](https://obsidian.md/) vault for the project is located at ```obsidian-vault-paintball```.

> Note that the .md files can be read through any text editor or Markdown viewer.

The vault contains Game Design documentation.
